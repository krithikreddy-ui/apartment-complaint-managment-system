ResiSolve is a simple web-based complaint management system designed for apartment communities. It allows residents to report maintenance issues and track their status in a transparent way. The system helps manage common problems like water leakage, power failures, lift malfunctions, and hygiene issues.

The application provides a clean user interface where users can log in, submit complaints, and view recently reported issues.

Features
1. User Login

Secure login interface for residents.

Validates username and password before accessing the system.

2. Complaint Submission

Residents can submit complaints with:

Flat Number

Issue Category

Urgency Level (Low / Medium / High)

Detailed Description

3. Complaint Categories

The system supports multiple complaint types:

Water Leakage

Power Failure

Lift Malfunction

Garbage / Hygiene

Noise Complaint

Other issues

4. Complaint Tracking

Users can:

View recently submitted complaints

Track complaint status (Open / Resolved)

Filter complaints by status

5. Dashboard Metrics

The sidebar shows:

Total Open Issues

Total Resolved Issues

6. Theme Toggle

Light/Dark mode switch for better user experience.

7. Clear Resolved Complaints

Option to remove resolved complaints from the list.
